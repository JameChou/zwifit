Enter the discrete steps that make up the route, one per line. Each step can be based on the current run’s distance or time. Enter the distance, a colon, then what should happen. Multiple actions can be taken, simply comma separate them. Time should be formatted as HH:MM:SS. Speeds can be in MPH or KPH.
1 Mile: 1% Incline
2.6 Miles: 6.2 MPH
0:30:00: 10% Incline
5 Miles: 2% Incline, 4 MPH
